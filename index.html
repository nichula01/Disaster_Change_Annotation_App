Help & Instructions

What is this tool?

This tool helps you correct flood/change masks generated by AI foundation models. The AI gives an approximate mask — your job is to fix false positives and false negatives.

Mask Convention

White / 255 = Flood or Change
Black / 0 = No flood / No change / Background

Step-by-step Workflow

  1. Load the ON/FLOOD image — this is the main base image.
  2. Load the BEFORE image — used for visual comparison.
  3. Load the FM Approx. Mask — the AI's prediction, becomes your starting mask.
  4. The red overlay shows the current flood prediction.
  5. Use Add Flood (A) to paint missed flood areas (adds white).
  6. Use Remove Flood (R) to erase wrongly predicted areas (adds black).
  7. Toggle BEFORE overlay to compare before/after the event.
  8. Enable Correction Diff to review what you changed.
  9. Export the Final Corrected Mask when satisfied.

Tools

  • Add Flood (A) — Paints WHITE = flood/change pixels
  • Remove Flood (R) — Paints BLACK = no-change pixels
  • Pan (P) — Drag to move the view
  • Scroll wheel — Zoom in/out centered on cursor

Correction Diff Colors

  • ■ Green — You added flood (AI missed it)
  • ■ Blue — You removed flood (AI was wrong)
  • ■ Yellow — Unchanged flood (AI was correct, you kept it)
  • Transparent — No flood in either mask
